# 产品需求文档（简版）

## 1. 产品定位

Interview Agent 是一个面向求职训练场景的智能面试问答系统。用户可以上传自己的面试资料、技术笔记、八股文、真实面试复盘与外部参考资料，系统通过 RAG 技术构建用户私有知识库，并在此基础上提供问答、抽题、模拟面试和训练反馈能力。

## 2. 核心目标

- 让系统基于用户资料进行可追溯的 RAG 问答
- 支持引用来源展示与可解释回答
- 支持后续题库生成、模拟面试和训练分析

## 3. 核心功能

### 3.1 文档知识库

- 支持上传 `txt / md / pdf`
- 文档解析、切分、向量化并写入 PGVector
- 支持知识库维度隔离检索

### 3.2 RAG 问答

- 用户围绕知识库提问
- 系统在知识库范围内检索相关 chunk
- 系统基于检索上下文生成回答
- 返回引用来源、检索步骤和调试信息

### 3.3 题库生成与训练

- 根据文档内容生成结构化面试题
- 支持模拟面试、回答评分、追问与总结

## 4. 当前实现重点

当前项目已经重点落地：

- 文档异步入库
- PGVector 向量存储
- metadata 增强检索
- query routing
- rerank
- 可解释问答
- 无证据受控 fallback

## 5. 当前风险与后续优化

- RAG 效果仍依赖文档质量与 chunk 策略
- 仍需继续强化混合检索、rerank 与评估体系
